What is Salesforce Service type setting?

In Salesforce Service, you can set the type of cases. Such type setting enables you to group cases, and other actions based on it.

How does GreenflyAI automate case type setting in Salesforce Service?

We use language identification powered by AI. This enables GreenflyAI to automatically set the type of cases in Salesforce Service.

All you need to do is show GreenflyAI a few examples of the cases you'd like us to set the type of.

For every new case, we analyse the vocabulary to identify the language of a given text. We can then automatically set the type of cases in Salesforce Service.

You could use automated case type setting in Salesforce Service to:

  • You could set the type of cases based on language identification.

  • You can route cases in Salesforce Service based on type setting.

  • You can use it to gain insights in Salesforce Service reporting.
